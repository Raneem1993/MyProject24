Mein Project hat Frontend und Backend . Frontend wurde in Angular entwickelt und das Backend mit Node.js.Die Anwendung geht es um eine "Nachhilfe" Webseite, wo man sich einloggen oder sich registrieren kann.<img width="1433" alt="Bildschirm­foto 2024-03-26 um 19 50 05" src="https://github.com/Raneem1993/MyProject24/assets/92716308/99ea6e09-b8ce-4f07-b3b6-7e39d605eb90">
<img width="1433" alt="Bildschirm­foto 2024-03-26 um 19 50 05" src="https://github.com/Raneem1993/MyProject24/assets/92716308/2d8af706-0881-4871-ba03-a9306f880887">
Wenn man auf den "Anmeldung" Button drückt, wird man einen Anmelgungsformular bekommen.Wenn der "Nutzer" schon im Datenbank registriert ist dann wurde auf die Starseite weitergeleitet.
Aber wenn der "Nutzer" noch nicht im Datenbank registriert ist, dann bekommt man einen Registrierungsformular.
